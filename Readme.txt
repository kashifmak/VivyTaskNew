Features of the APP

1. Search field to search for the doctors.
2. Mic input to enable the user to search the doctor by giving voice input.
3. Internet connectivity check, and inform the users if not connected to internet.
4. Detail screen to view more information of each doctor including (rating, email, website and opening hours)
5. Allow the user to click the website link to visit the website of the doctor
6. Some screenshots are added inside 'Screenshots' folder
7. Added apk to download and install


Note
1. After cloning if Android studio says “Please select Android SDK” then please goto File -> Sync Project with Gradle Files
as described here https://stackoverflow.com/questions/34353220/android-studio-please-select-android-sdk
